# Casos de Uso

## Gestión de Productos

Como administrador

Quiero crear productos

Para que aparezcan en el catálogo.

---

Como administrador

Quiero editar productos

Para mantener información actualizada.

---

Como administrador

Quiero eliminar productos

Para retirar artículos del catálogo.

---

## Gestión de Categorías

Como administrador

Quiero administrar categorías

Para organizar los productos.

---

## Configuración de Tienda

Como administrador

Quiero modificar la información del negocio

Para personalizar la tienda.

---

Como administrador

Quiero cambiar colores y logotipo

Para adaptar la apariencia al negocio.

---

## Cliente

Como cliente

Quiero consultar productos

Para encontrar artículos de interés.

---

Como cliente

Quiero buscar productos

Para localizar artículos rápidamente.

---

Como cliente

Quiero agregar productos al carrito

Para preparar una compra.
